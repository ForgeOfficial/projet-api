# <a name="Utilisation">Utilisation:</a>
##### 1. Installation
```bash
git clone https://github.com/ForgeOfficial/projet-api.git
cd projet-api
```
###### a. Backend
```bash
npm install
```
```bash
npm run start
```

###### b. Frontend
Dans un deuxieme terminal:
```bash
cd frontend
```
```bash
npm install
```
```bash
npm run dev
```

##### 2. Acceder au site
http://localhost:3001
